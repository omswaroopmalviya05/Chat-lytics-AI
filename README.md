# 🚀 Chatlytics AI

### Conversation Analytics Platform

Chatlytics AI is a modern analytics dashboard that transforms exported chat data into meaningful insights using data visualization, behavioral analytics, and smart metrics.

Built with Python, Streamlit, Pandas, Plotly, and NLP-based text analysis.

## 📌 Features

### 📊 Analytics Dashboard
- Total messages, words, media, links shared
- Peak active hour detection
- Average messages per day
- Most active users ranking

### 📈 Trend Analysis
- Monthly timeline
- Daily timeline
- Weekly activity heatmap
- Most active day / month

### 👥 User Insights
- Individual user filtering
- User contribution percentage
- Message activity comparison

### 🧠 NLP & Text Intelligence
- Word cloud generation
- Most common words
- Emoji usage analysis
- Smart summary insights

### 🔒 Privacy First
- Fully local processing
- No external database required
- Chat file analyzed securely
